A Unmanned Dining System for small restaurant.  一个为小微餐馆设计的具有点餐接单管理统计功能无人点餐系统。

>这是作者第一个github练手项目
>> this is the first project in my github

- 架构
    1. 前端：Angular
    2. 后端：SpringBoot
    3. 其他中间件：MariaDB，RabbitMq，Redis
    
- 模块
    1. 基础数据模块
    2. 下单模块
    3. 接单制作模块
    4. 订单管理模块 
    5. 基础报表模块
    6. 增值服务模块
        * 报表图标模块
        * 优惠活动模块
        * 会员模块
        * 皮肤模块
